# 🎮 Mini Games Hub

A modern, interactive **Mini Games Hub web application** built using only **HTML, CSS, and Vanilla JavaScript**.  
This project brings together multiple classic games in one smooth, animated, and responsive dashboard.

## ✨ Features

### 🏠 Game Hub Dashboard
- Clean modern UI with animated game cards
- Smooth transitions between games
- Responsive layout (mobile + desktop)
- Hover animations and interactive UI effects

---

### ❌ Tic Tac Toe
- Two-player gameplay (X vs O)
- Win/draw detection
- Highlight winning combination
- Smooth reset animation

---

### ✊✋✌️ Rock Paper Scissors
- Player vs Computer mode
- Animated selections
- Live score tracking
- Result animations (Win / Lose / Draw)

---

### 🔢 Guess the Number
- Random number (1–100)
- Smart hints: Too High / Too Low
- Attempt counter
- Win animation with confetti effect 🎉

---

### 🧠 Memory Card Flip Game
- Flip card animations (3D effect)
- Match pair detection
- Move counter
- Smooth match/mismatch feedback

---

## 🎨 UI/UX Highlights
- Modern dark-themed design (light mode optional)
- Glassmorphism / soft UI effects
- Smooth page transitions
- Micro-interactions (hover, click, bounce effects)
- Fully responsive design

---

## ⚙️ Technologies Used
